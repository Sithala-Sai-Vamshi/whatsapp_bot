Rayeva AI WhatsApp Support Bot

This static page is published via GitHub Pages for architecture visibility.

Architecture

WhatsApp Cloud API -> Webhook Receiver -> BullMQ Queue -> Message Processor
                                |                    |-> Intent Classification
                                |                    |-> Business Rules (orders/policies)
                                |                    |-> AI Decision Engine (structured JSON)
                                |                    |-> Escalation Service
                                -> PostgreSQL Logs + Redis Context + WhatsApp Response

Structured AI Output Contract

All model responses are validated against:

{
  "customer_message": "string",
  "confidence": 0.0,
  "requires_escalation": false,
  "escalation_reason": null,
  "follow_up_question": null,
  "cited_facts": []
}
